## 六、选购避坑与下单注意事项

**坑1：MaxMind反欺诈拦截订单**
ZgoCloud开了WHMCS MaxMind自动检测，下单时IP地址、电话号码、所选国家必须自洽。比如你用美国IP下单，电话却填+86中国号、地址写上海——大概率被判定Fraud订单卡住。解决办法：三者保持一致就行，不要求信息真实，但要"看起来像同一个地方的人"。

**坑2：特价方案不可退款**
所有Specials系列官方明确写了"No refunds/money back on those plans"，下手前确认自己真的要这台机器再买。常规方案退款政策以官网Terms of Service为准。

**坑3：IIJ/BGP/国际线路不支持以"线路不优化"为由退款**
大阪IIJ线路、洛杉矶Global国际线路官方明确写明"not optimized for China, and refunds cannot be requested for this reason"——买之前先看Looking Glass（`https://lg.naaldwijk.nl.zgovps.com/`）测路由，确认延迟和路由可接受再下单。

**坑4：双ISP IP不是住宅IP**
双ISP IP是数据中心托管IP，除IP2Location外的IP库都识别为dual ISP，但本质还是机房IP——做账号注册能起到一定规避作用，但跟真正的住宅IP是两回事，不可因此申请退款。

**坑5：优惠码适用范围有限**
`8NU44CM6LZ`和`BPZZ1GE8T7`只对**常规套餐年付**生效，特价方案、双ISP系列、香港BGP特价等系列不支持。结账前先看购物车"Use promotional code"输入框下方是否有提示。

**坑6：流量是Fair Use不是无限**
所有套餐的流量都标注"Fair Use"或"基于公平使用原则"——意味着超过月流量上限后不会立即停机，但可能限速或要求补流量费。监控好流量使用，避免月底突然被限速。

## 八、常见问题速答

**Q1：ZgoCloud和ZgoVPS是同一家吗？**
是的，ZgoCloud是品牌名，ZgoVPS是早期叫法，注册主体都是ZgoShop, Inc.（特拉华州备案6298021），客户端域名是`clients.zgovps.com`，官网是`zgovps.com`，两个名字指的是同一家服务商。

**Q2：优惠码过期了怎么办？**
`8NU44CM6LZ`目前多个渠道显示有效期到本年7月底，`BPZZ1GE8T7`的具体过期时间需要以下单页校验为准。过期后通常会有新码放出，可以关注ZgoCloud的Telegram频道或者定期回看本文更新。

**Q3：可以用优惠码买特价方案吗？**
不行。特价方案本身已是底价，不能叠加优惠码。如果你看到购物车里特价方案能用码，多半是系统异常，下单成功后也可能被取消。

**Q4：续费价格会涨吗？**
循环优惠码续费同价，特价方案续费是否同价需要看具体套餐说明——部分Specials是首年特价、续费原价，下单前最好先提交工单问客服确认续费政策。

**Q5：洛杉矶优化线路和Premium优化线路区别在哪？**
优化线路（9929&CMIN2）走的是AS9929和CMIN2优质回程，三网优化但不上CN2 GIA；Premium优化线路（CN2 GIA&9929&CMIN2）多走一条CN2 GIA，电信回程更稳更快——所以同配置Premium比优化贵约30%。预算够选Premium，预算紧优化线路也够用。
